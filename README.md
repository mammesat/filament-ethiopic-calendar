# Filament Ethiopic Date Picker

A **Filament v5 plugin** that provides Ethiopic calendar support with automatic conversion between Ethiopic and Gregorian dates.

---

## 🚀 Features

- Ethiopic calendar input for Filament forms
- Automatic conversion to Gregorian (`Y-m-d`) for database storage
- Conversion back to Ethiopic for display
- Built on Filament v5 + Livewire v4
- Clean, extensible architecture (no core overrides)

---

## 📦 Installation

```bash
composer require mammesat/filament-ethiopic-date-picker
